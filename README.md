## 📖 About

This repository is my personal space for learning **Java** from the ground up. It's a collection of small programs, experiments, and notes as I work through the fundamentals of the language.

I'm using this project to get comfortable with core concepts, build good habits, and track how far I've come.

## 🚀 Getting Started

To run any of the programs in this project:

```bash
# Navigate to the source folder
cd src

# Compile a Java file
javac FileName.java

# Run the compiled program
java FileName
```

> 💡 If you're using **IntelliJ IDEA**, you can simply open the project and hit the green **Run** button.
